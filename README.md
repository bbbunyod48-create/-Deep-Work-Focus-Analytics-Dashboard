🧠 Deep Work & Focus Analytics Dashboard
Zamonaviy SaaS uslubidagi bu dashboard foydalanuvchilarning ishlash unumdorligini o'lchash, chuqur fokuslash vaqtlarini kuzatish va real-vaqtda tahlil qilish uchun mo'ljallangan. Pomodoro texnikasi asosida ishlaydi.

ReactViteTailwindCSSZustand

✨ Asosiy Xususiyatlar (Core Features)
🚀 Advanced Pomodoro Engine: Estetik va silliq SVG circular progress taymer. Fokus, Qisqa Tanaffus va Uzoq Tanaffus vaqtlarini moslashtirish imkoniyati.
🧘 Deep Work Mode: Barcha chalg'ituvchi elementlarni yashirib, faqat taymerni ekranga chiqaradigan "Chuqur Ish" rejimi.
🏷️ Session Tagging: Har bir sessiyani 'Kodlash', 'Dizayn', 'Uchrashuv' kabi teglar bilan belgilash.
📊 Data Visualization: Recharts yordamida keng qamrovli analitika:
So'nggi 7 kunlik fokus soatlarini ko'rsatuvchi Bar Chart.
Teglar bo'yicha vaqt taqsimotini ko'rsatuvchi Pie Chart.
Fokus intensivligini aks ettiruvchi Line Graph.
💾 Persistence: Zustand va localStorage yordamida barcha sessiya tarixini saqlash (sahifa yangilansa ham ma'lumot yo'qolmaydi).
🌙 High-end Dark Theme: Slate va Emerald ranglari asosidagi professionall dark dizayn.
📱 Responsive Design: Qurilmaga moslashuvchi va yig'iladigan (collapsible) sidebar.
🛠️ Texnologiyalar (Tech Stack)
Frontend: React 18+ (Vite)
Styling: Tailwind CSS
State Management: Zustand
Charts: Recharts
Icons: Lucide React
